Windows:
	- download python from https://www.python.org/downloads/ and install it
	- run install-django.bat batch file to install depencies
	- run "run.bat" to start server
	- open browser and go 127.0.0.1

Linux 
	- run install-django.sh as root to install depencies
	- run "run.sh" to start server
	- open browser and go 127.0.0.1
